# IDBI Innovate 2026 – POC / Idea Submission

## Track 01: Digital Wealth Management

### Project Name: **FinSight AI – Dhan Sakhi**

> AI-Powered Avatar-Based Multilingual Wealth Advisor

---

## 1. Brief About the Idea

**FinSight AI (Dhan Sakhi)** is an AI-powered, avatar-based digital wealth management platform that integrates directly into IDBI Bank's mobile application. It delivers personalized, scalable wealth advisory services through an intuitive conversational interface powered by a photorealistic AI avatar.

Instead of navigating complex menus or reading dense financial jargon, customers simply **talk** to their personal AI wealth advisor — in Hindi, Tamil, Telugu, Bengali, Marathi, or any of 12+ Indian languages. The avatar understands context, remembers past interactions, analyzes portfolios in real-time, and provides SEBI-compliant investment recommendations.

The system combines:
- **Generative AI** for financial reasoning and natural conversation
- **Agentic AI** with specialized agents (Research, Compliance, Execution, Portfolio)
- **Conversational AI** for multilingual voice interaction
- **Computer Vision** for lifelike avatar animation with lip-sync and expressions
- **Machine Learning** for portfolio optimization and risk profiling

This is not a chatbot. This is a **digital relationship manager** — available 24/7, infinitely patient, deeply knowledgeable, and fluent in every Indian language.

## 6. List of Features

### A. Customer-Facing Features

| Category | Features |
|----------|----------|
| **Conversation** | Natural voice interaction, multilingual (12+ languages), context memory across sessions, follow-up questions |
| **Avatar** | Photorealistic rendering, lip-sync, facial expressions, emotion-aware responses, customizable appearance |
| **Portfolio Management** | Real-time portfolio view, asset allocation analysis, performance attribution, benchmark comparison |
| **Investment Advisory** | Goal-based planning, SIP recommendations, MF comparison, tax optimization, rebalancing alerts |
| **Risk Assessment** | Behavioral risk profiling (from transactions), dynamic risk score, stress-test scenarios |
| **Financial Planning** | Retirement planning, child education, home purchase, emergency fund, insurance gap analysis |
| **Education** | Concept explanations in simple language, personalized examples using customer's own data |
| **Execution** | One-click SIP setup, fund switch, lumpsum investment, all within conversation |
| **Alerts & Monitoring** | Market impact alerts, goal deviation warnings, rebalancing triggers, tax-saving deadlines |

### B. Agentic AI Features (Backend Agents)

| Agent | Role |
|-------|------|
| **Research Agent** | Fetches real-time market data, fund performance, sector analysis before responding |
| **Compliance Agent** | Validates every recommendation against SEBI suitability norms and customer risk profile |
| **Portfolio Agent** | Analyzes current holdings, computes metrics, identifies gaps and concentration risks |
| **Execution Agent** | Places orders, modifies SIPs, processes switches upon customer approval |
| **Monitoring Agent** | Continuous background monitoring of portfolio health and market events |
| **Personalization Agent** | Maintains customer preferences, adapts communication style, remembers context |

### C. Bank Operations Features

| Feature | Description |
|---------|-------------|
| Analytics Dashboard | Customer engagement metrics, AUM growth, conversion tracking |
| RM Escalation | Seamless handoff to human RM when customer requests or complexity requires |
| Campaign Integration | Targeted offers based on AI-identified opportunities |
| Compliance Audit Trail | Complete log of every recommendation with reasoning for regulatory audits |
| A/B Testing | Test different advisory strategies and measure conversion impact |
